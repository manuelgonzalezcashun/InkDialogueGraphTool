## Installation

1. Open Unity Package Manager -> Add package from git URL...
2. Install the **Ink Unity Integration** first: `https://github.com/inkle/ink-unity-integration.git#upm` (This will help with compiling the Ink Files inside the unity project)
3. Install this **Dialogue Graph Tool**: `https://github.com/manuelgonzalezcashun/InkDialogueGraphTool.git`
